## Quick Start

```bash
# Install dependencies
npm install

# Start the game
npm run dev
```

Open http://localhost:5173 in your browser to play!

## How to Play

1. Click any square to place your mark (X goes first)
2. Take turns with your opponent
3. Get three in a row to win!
